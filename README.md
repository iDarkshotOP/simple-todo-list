# Simple To-Do List App

![Python Version](https://img.shields.io/badge/python-3.x-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

A basic console-based To-Do List application in Python. Add tasks, view the task list, and mark tasks as completed.

## Features

- Add tasks to the to-do list with priorities.
- View the list of tasks and their completion status.
- Mark tasks as completed.

## Usage

1. Run the `main.py` script.
2. Choose from the available options to add tasks, view tasks, mark tasks as completed, or quit the app.
